# Wallet Generator REST-API


## How to start

-   `yarn install`
-   `yarn start`

## How to works

**Generate a new Wallet**

-   http://localhost:3333/wallet/new

**Generate a Wallet from Seed**

-   http://localhost:3333/wallet/from/your,seed,separated,by,commas
